# DronePoint ESP Script

Ультимативный скрипт для Roblox плейса **DronePoint**. Этот скрипт позволяет подсвечивать (ESP) различные объекты в игре, такие как FPV-дроны, Шахеды и ракеты.

## Особенности
- **Интерактивный GUI**: Современное меню для управления всеми функциями в реальном времени.
- **Highlight ESP**: Использует встроенную систему `Highlight` для четкой видимости сквозь стены.
- **Smart Filtering**: Автоматически определяет типы угроз (FPV, Shahed, Missile).
- **Draggable UI**: Меню можно перемещать по экрану.
- **Optimization**: Не нагружает процессор, работает плавно.

## Использование
Скопируйте код из `src/init.lua` и вставьте его в ваш исполнитель (executor) скриптов (например, Synapse Z, Wave, Solara и т.д.).

## Конфигурация
В начале скрипта `src/init.lua` вы можете изменить настройки:
- `HighlightColor`: Цвет подсветки.
- `FillOpacity`: Прозрачность заливки.
- `OutlineOpacity`: Прозрачность контура.
